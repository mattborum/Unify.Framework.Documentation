Creating and Using Unify Enumerators

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

To create a Unify Enumerator instance, use the new() operator and pass the source collection to the constructor. The source collection can be any class that implements either of Unify's IUnifyEnumerable<T> or IUnifyEnumerable interface types, or either of .NET's IEnumerable<T> or IEnumerable interface types; this includes any single-dimensional, multidimensional or jagged array.

A reference to a Unify Enumerator can be held in either; the concrete type of the Unify Enumerator, either of Unify's IUnifyEnumerator<T> or IUnifyEnumerator interface types, or either of .NET's IEnumerator<T> or IEnumerator interface types:

IList<string> list = new List<string> { "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Mango" };

// Example of matching concrete types
UnifyEnumerator<string> defaultGeneric = new UnifyEnumerator<string>(list);
UnifyYoYoEnumerator<string> yoyoGeneric = new UnifyYoYoEnumerator<string>(list);
UnifyRandomEnumerator<string> randomGeneric = new UnifyRandomEnumerator<string>(list);

// Example of Unify Interface types
IUnifyEnumerator<string> defaultGenericInterface = new UnifyYoYoEnumerator<string>(list);
IUnifyEnumerator defaultNonGenericInterface = new UnifyRandomEnumerator<string>(list);

// Example of .NET interface types
IEnumerator<string> dotNetGenericHoldingUnifyGeneric = new UnifyEnumerator<string>(list);
IEnumerator dotNetNonGenericHoldingUnifyGeneric = new UnifyRandomEnumerator<string>(list);

* Using .NET's IEnumerator<T> or IEnumerator interface types to hold references to Unify Enumerators will limit functionality. See IUnifyEnumerator.


When a Unify Enumerator is created, or after the Reset() method is called, the Unify Enumerator is positioned before the element in the collection specified by the startIndex property of the Enumerable Range. The first call to the MoveNext() method moves the Unify Enumerator to the first element in the collection's Enumerable Range. The MoveNext() method returns true while the Unify Enumerator is pointing to a valid element; otherwise, false if the end of the enumeration has been reached or the Unify Enumerator is invalid.

The Unify Enumerator's Current property is used to access the object the Unify Enumerator is positioned at, this is of type T for the generic IUnifyEnumerator<T> type, and of type object for the non-generic IUnifyEnumerator type. The Current property returns the same value until the MoveNext() method is called, then Current is set to the next element specified by the Enumerable Range.

To use the Unify Enumerator again, call the Reset() method.

Example

This example demonstrates each of the three built-in Unify Enumerator types; UnifyEnumerator<T>, UnifyYoYoEnumerator<T> and UnifyRandomEnumerator<T>, with a generic type T of string, to iterate over a .NET List<string> containing the words; "Say", "Hello", "To", "Unify", "Enumerators". Unify's IUnifyEnumerator<T> type is used to hold a reference to each of the Unify Enumerators.

Attach the script below to a GameObject and press:

  • S to Start the Standard Enumerator
  • Y to Start the Yo-Yo Enumerator
  • R to Start the Random Enumerator
  • X to Reset all Enumerators
using System;
using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_CreateAndUse : MonoBehaviour
{
    private IUnifyEnumerator<string> unifyStandardEnumerator;
    private IUnifyEnumerator<string> unifyYoyoEnumerator;
    private IUnifyEnumerator<string> unifyRandomEnumerator;
    private IList<string> list;

    void Start()
    {
        list = new List<string> { "Say", "Hello", "To", "Unify", "Enumerators" };

        unifyStandardEnumerator = new UnifyEnumerator<string>(list);
        unifyYoyoEnumerator = new UnifyYoYoEnumerator<string>(list);
        unifyRandomEnumerator = new UnifyRandomEnumerator<string>(list);

        print("press S for STANDARD, Y for YO-YO, R for RANDOM, X to RESET ALL ENUMERATORS");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            RunStandard();
        }

        if (Input.GetKeyUp(KeyCode.Y))
        {
            RunYoYo();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            RunRandom();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            ResetAll();
        }
    }

    private void RunStandard()
    {
        print("--- Standard Enumerator ---");
        while (unifyStandardEnumerator.MoveNext())
        {
            print(unifyStandardEnumerator.Current);
        }
        print(Environment.NewLine);
    }

    private void RunYoYo()
    {
        print("--- Yo-Yo Enumerator ---");
        while (unifyYoyoEnumerator.MoveNext())
        {
            print(unifyYoyoEnumerator.Current);
        }
        print(Environment.NewLine);
    }

    private void RunRandom()
    {
        print("--- Random Enumerator ---");
        while (unifyRandomEnumerator.MoveNext())
        {
            print(unifyRandomEnumerator.Current);
        }
        print(Environment.NewLine);
    }

    private void ResetAll()
    {
        print("--- Resetting Enumerators ---");
        print(Environment.NewLine);
        unifyStandardEnumerator.Reset();
        unifyYoyoEnumerator.Reset();
        unifyRandomEnumerator.Reset();
    }
}
