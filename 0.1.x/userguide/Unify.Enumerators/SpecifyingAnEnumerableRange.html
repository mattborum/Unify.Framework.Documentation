Specifying an Enumerable Range

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Unify Enumerators can iterate over a partial collection. This functionality is provided by Unify's Enumerable Range feature, which is encapsulated in the UnifyEnumerableRange and UnifyEnumerableRange<T> classes. Unify Enumerators expose this functionality through the enumerableRange property.

An enumerable range is defined by a start item and an end item; these values are initialised by their index position through the startIndex and endIndex properties. As the underlying collection is modified the UnifyEnumerableRange will adjust so it is always defined by the items between, and including, the initial start and end items. If either of the start or end items are removed, the new start or end item will become the next item inside the range. When all items in the range have been removed the UnifyEnumerableRange cannot be used further.

To specify an enumerable range that always starts from the first item of a collection, or always ends at the last item of a collection, regardless of any changes made, set the startIndex and/or endIndex properties to -1. This is the default setting.

Example

This example demonstrates the four Unify Enumerator types, with a generic type T of string, to demonstrate iterating over an enumerable range. The example iterates over a list of fruits; "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry" and "Mango", with the enumerable range having a startIndex of 2 and an endIndex of 8, which includes all items between, and including, "Banana" and "Cherry".

Attach the script below to a GameObject and press:

  • F to Start the Forwards Enumerator
  • B to Start the Backwards Enumerator
  • Y to Start the Yo-Yo Enumerator
  • R to Start the Random Enumerator
  • X to Reset all Enumerators
using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_SpecifyingAnEnumerableRange : MonoBehaviour
{
    private IList<string> list;
    private IUnifyEnumerator<string> forwards;
    private IUnifyEnumerator<string> backwards;
    private IUnifyEnumerator<string> yoyo;
    private IUnifyEnumerator<string> random;

    void Awake()
    {
        list = new List<string> { "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Mango" };

        forwards = new UnifyForwardsEnumerator<string>(list);
        forwards.enumerableRange.startIndex = 2; // range starts at "Banana"
        forwards.enumerableRange.endIndex = 8; // range ends at "Cherry"

        backwards = new UnifyBackwardsEnumerator<string>(list);
        backwards.enumerableRange.startIndex = 2;
        backwards.enumerableRange.endIndex = 8;

        yoyo = new UnifyYoYoEnumerator<string>(list);
        yoyo.enumerableRange.startIndex = 2;
        yoyo.enumerableRange.endIndex = 8;

        random = new UnifyRandomEnumerator<string>(list);
        random.enumerableRange.startIndex = 2;
        random.enumerableRange.endIndex = 8

        print("press F for FORWARDS, B for BACKWARDS, Y for YO-YO, R for RANDOM, X to RESET ALL ENUMERATORS");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.F))
        {
            RunForwards();
        }

        if (Input.GetKeyUp(KeyCode.B))
        {
            RunBackwards();
        }

        if (Input.GetKeyUp(KeyCode.Y))
        {
            RunYoYo();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            RunRandom();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            ResetAll();
        }
    }

    private void RunForwards()
    {
        print("--- Forwards Enumerator ---");
        while (forwards.MoveNext())
        {
            print(forwards.Current);
        }
    }

    private void RunBackwards()
    {
        print("--- Backwards Enumerator ---");
        while (backwards.MoveNext())
        {
            print(backwards.Current);
        }
    }

    private void RunYoYo()
    {
        print("--- Yo-Yo Enumerator ---");
        while (yoyo.MoveNext())
        {
            print(yoyo.Current);
        }
    }

    private void RunRandom()
    {
        print("--- Random Enumerator ---");
        while (random.MoveNext())
        {
            print(random.Current);
        }
    }

    private void ResetAll()
    {
        print("--- Resetting Enumerators ---");
        forwards.Reset();
        backwards.Reset();
        yoyo.Reset();
        random.Reset();
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Setting the startIndex property to a value of less than -1 will throw an ArgumentOutOfRangeException
  • Setting the endIndex property to a value of less than -1 will throw an ArgumentOutOfRangeException
