Unify Enumerators

Namespace: Unify.Collections
Assembly: Unify.Collections.dll

Unify offers four ways to iterate over a collection; forwards, backwards, yo-yo and random. Each enumerator has a generic and non-generic version, available through the following classes:

  • UnifyForwardsEnumerator<T> and UnifyForwardsEnumerator
  • UnifyBackwardsEnumerator<T> and UnifyBackwardsEnumerator
  • UnifyYoYoEnumerator<T> and UnifyYoYoEnumerator
  • UnifyRandomEnumerator<T> and UnifyRandomEnumerator

To create a Unify Enumerator instance, use the new() operator and pass the source collection to the constructor. The source collection can be any class that implements either of .NET's IEnumerable or IEnumerable<T> interfaces, this includes any single-dimensional, multidimensional or jagged array, or any class that implements either of Unify's IUnifyList or IUnifyList<T> interfaces.

You can hold a reference to a Unify Enumerator in either; a concrete type matching the Unify Enumerator you are creating or either of Unify's IUnifyEnumerator and IUnifyEnumerator<T> interface types:

IList<string> list = new List<string> { "one", "two", "three", "four", "five", "six", "seven", "eight", "nine", "ten", "eleven", "twelve" };

UnifyForwardsEnumerator forwards = new UnifyForwardsEnumerator(list);
IUnifyEnumerator forwardsInterface = new UnifyForwardsEnumerator(list);

UnifyBackwardsEnumerator<string> backwardsGeneric = new UnifyBackwardsEnumerator<string>(list);
IUnifyEnumerator<string> backwardsGenericInterface = new UnifyBackwardsEnumerator<string>(list);

When a Unify Enumerator is created or after the Reset() method is called, the Unify Enumerator is positioned before the first element of the collection, the first call to the MoveNext() method moves the Unify Enumerator to the first element of the collection. The MoveNext() method returns true while the Unify Enumerator has another element to move to; otherwise, false if the end of the collection has been reached.

To access the object at which the Unify Enumerator is positioned, use the Current property. The Current property is of type object for a non-generic Unify Enumerator or of type T for a generic Unify Enumerator and returns the same object until MoveNext() is called. MoveNext() sets Current to the next element:

IList<string> list = new List<string> { "one", "two", "three", "four", "five", "six", "seven", "eight", "nine", "ten", "eleven", "twelve" };
UnifyBackwardsEnumerator<string> backwards = new UnifyBackwardsEnumerator<string>(list);

while (backwards.MoveNext())
{
    print(backwards.Current);
}

To use the Unify Enumerator again, call the Reset() method:

IList<string> list = new List<string> { "one", "two", "three", "four", "five", "six", "seven", "eight", "nine", "ten", "eleven", "twelve" };
UnifyBackwardsEnumerator<string> backwards = new UnifyBackwardsEnumerator<string>(list);

while (backwards.MoveNext())
{
    print(backwards.Current);
}

print("Resetting enumerator...");

backwards.Reset();

while (backwards.MoveNext())
{
    print(backwards.Current);
}
