Creating an Enumerable Range

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

To create an Enumerable Range, use the new() operator to create an instance of the UnifyEnumerableRange class. A single public constructor is provided to pass in values for the properties; startIndex, endIndex, loopCount, step and locked. All parameters are optional. When a parameter is not specified, the default value is used:

  • UnifyEnumerableRange.AlwaysFirstIndex
  • UnifyEnumerableRange.AlwaysLastIndex
  • UnifyEnumerableRange.DefaultLoopCount
  • UnifyEnumerableRange.DefaultStep
  • UnifyEnumerableRange.DefaultLocked
