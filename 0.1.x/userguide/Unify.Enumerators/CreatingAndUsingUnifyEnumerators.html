Creating and Using Unify Enumerators

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

To create a Unify Enumerator instance, use the new() operator and pass the source collection to the constructor. The source collection can be any class that implements either of Unify's IUnifyEnumerable or IUnifyEnumerable<T> interface types, or either of .NET's IEnumerable or IEnumerable<T> interface types, this includes any single-dimensional, multidimensional or jagged array.

A reference to a Unify Enumerator can be held in either; the concrete type matching the Unify Enumerator type, either of Unify's IUnifyEnumerator and IUnifyEnumerator<T> interface types, either of .NET's IEnumerator and IEnumerator<T> interface types; also a Unify or .NET's non-generic enumerator interfaces can hold a reference to a generic Unify Enumerator:

IList<string> list = new List<string> { "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Mango" };

// Matching concrete types
UnifyForwardsEnumerator<string> forwardsGeneric = new UnifyForwardsEnumerator<string>(list);

// Unify Interface types
IUnifyEnumerator<string> backwardsGenericInterface = new UnifyBackwardsEnumerator<string>(list);
IUnifyEnumerator foo = new UnifyRandomEnumerator<string>(list);

// .NET interface types
IEnumerator<string> dotNetGenericHoldingUnifyGeneric = new UnifyBackwardsEnumerator<string>(list);
IEnumerator dotNetNonGenericHoldingUnifyGeneric = new UnifyRandomEnumerator<string>(list);

When a Unify Enumerator is created, or after the Reset() method is called, the Unify Enumerator is positioned before the first element of the collection, the first call to the MoveNext() method moves the Unify Enumerator to the first element of the collection. The MoveNext() method returns true while the Unify Enumerator is pointing to a valid element; otherwise, false if the end of the enumeration has been reached.

The Unify Enumerator's Current property is used to access the object the Unify Enumerator is positioned at, this is of type object for the non-generic IUnifyEnumerator type and of type T for the generic IUnifyEnumerator<T> type. The Current property returns the same value until the MoveNext() method is called, then Current is set to the next element.

To use the Unify Enumerator again, call the Reset() method.

Example

This example demonstrates each of the built-in Unify Enumerator types, with a generic type T of string, to iterate over a .NET List<string> containing the numbers 1 to 5 as words; "ONE", "TWO", "THREE", "FOUR" and "FIVE".

Attach the script below to a GameObject and press:

  • F to Start the Forwards Enumerator
  • B to Start the Backwards Enumerator
  • Y to Start the Yo-Yo Enumerator
  • R to Start the Random Enumerator
  • X to Reset all Enumerators
using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_CreateAndUse : MonoBehaviour
{
    private UnifyForwardsEnumerator<string> forwards;
    private UnifyBackwardsEnumerator<string> backwards;
    private UnifyYoYoEnumerator<string> yoyo;
    private UnifyRandomEnumerator<string> random;
    private IList<string> list;

    void Start()
    {
        list = new List<string> { "One", "Two", "Three", "Four", "Five" };

        forwards = new UnifyForwardsEnumerator<string>(list);
        backwards = new UnifyBackwardsEnumerator<string>(list);
        yoyo = new UnifyYoYoEnumerator<string>(list);
        random = new UnifyRandomEnumerator<string>(list);

        print("press F for FORWARDS, B for BACKWARDS, Y for YO-YO, R for RANDOM, X to RESET ALL ENUMERATORS");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.F))
        {
            RunForwards();
        }

        if (Input.GetKeyUp(KeyCode.B))
        {
            RunBackwards();
        }

        if (Input.GetKeyUp(KeyCode.Y))
        {
            RunYoYo();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            RunRandom();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            ResetAll();
        }
    }

    private void RunForwards()
    {
        print("--- Forwards Enumerator ---");
        while (forwards.MoveNext())
        {
            print(forwards.Current);
        }
    }

    private void RunBackwards()
    {
        print("--- Backwards Enumerator ---");
        while (backwards.MoveNext())
        {
            print(backwards.Current);
        }
    }

    private void RunYoYo()
    {
        print("--- Yo-Yo Enumerator ---");
        while (yoyo.MoveNext())
        {
            print(yoyo.Current);
        }
    }

    private void RunRandom()
    {
        print("--- Random Enumerator ---");
        while (random.MoveNext())
        {
            print(random.Current);
        }
    }

    private void ResetAll()
    {
        print("--- Resetting Enumerators ---");
        forwards.Reset();
        backwards.Reset();
        yoyo.Reset();
        random.Reset();
    }
}
