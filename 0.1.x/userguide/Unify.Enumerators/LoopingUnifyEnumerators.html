Looping Unify Enumerators

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Unify Enumerators provide a loopCount property to specify how many times an enumeration should be repeated. The value of the loopCount property should be a value greater than zero; the default value is 1.

Example

This example demonstrates each of the three built-in Unify Enumerator types; UnifyEnumerator<T>, YoYo<T> and Random<T>, with a generic type T of string, to iterate over a list containing the colours of the rainbow; "Red", "Orange", "Yellow", "Green", "Blue", "Indigo" and "Violet".

The standard UnifyEnumerator is used to iterate over the whole collection; this gives an single pass of the enumeration as; "Red", "Orange", "Yellow", "Green", "Blue", "Indigo", "Violet". The loopCount property is set to a value of 2.

The standard UnifyEnumerator is used again to iterate backwards over the collection with Enumerable Range values of 5 for the startIndex property and 1 for the endIndex property; this gives a single pass of the enumeration as; "Indigo", "Blue", "Green", "Yellow", "Orange". The loopCount property is set to a value 2.

The UnifyYoYoEnumerator is used to iterate in a reverse yo-yo fashion over an Enumerable Range defined by a startIndex value of 3 and an endIndex value of 0; this gives a single pass of the enumeration as; "Green", "Yellow", "Orange", "Red", "Orange", "Yellow", "Green". The loopCount property is set to a value of 2.

The UnifyRandomEnumerator is used with Enumerable Range values of 1 for the startIndex property and 5 for the endIndex property; this gives a single pass of the enumeration as a randomised list of the items; "Orange", "Yellow", "Green", "Blue", "Indigo". The loopCount property is set to 3.

Attach the script below to a GameObject and press:

  • S to Start the Standard Enumerator
  • B to Start the Standard Enumerator Backwards
  • Y to Start the Yo-Yo Enumerator
  • R to Start the Random Enumerator
  • X to Reset all Enumerators
using System;
using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_LoopingUnifyEnumerators : MonoBehaviour
{
    private IList<string> list;
    private IUnifyEnumerator<string> standardForwards;
    private IUnifyEnumerator<string> standardBackwards;
    private IUnifyEnumerator<string> yoyo;
    private IUnifyEnumerator<string> random;

    void Awake()
    {
        list = new List<string> { "Red", "Orange", "Yellow", "Green", "Blue", "Indigo", "Violet" };

        standardForwards = new UnifyEnumerator<string>(list, loopCount: 2);
        standardBackwards = new UnifyEnumerator<string>(list, startIndex: 5, endIndex: 1, loopCount: 2);
        yoyo = new UnifyYoYoEnumerator<string>(list, startIndex: 3, endIndex: 0, loopCount: 2);
        random = new UnifyRandomEnumerator<string>(list, startIndex: 1, endIndex: 5, loopCount: 3);

        PrintKeys();
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("--- Standard Enumerator ---");
            while (standardForwards.MoveNext())
            {
                print(standardForwards.Current);
            }
            print(Environment.NewLine);
            PrintKeys();
        }

        if (Input.GetKeyUp(KeyCode.B))
        {
            print("--- Backwards Enumerator ---");
            while (standardBackwards.MoveNext())
            {
                print(standardBackwards.Current);
            }
            print(Environment.NewLine);
            PrintKeys();
        }

        if (Input.GetKeyUp(KeyCode.Y))
        {
            print("--- YoYo Enumerator ---");
            while (yoyo.MoveNext())
            {
                print(yoyo.Current);
            }
            print(Environment.NewLine);
            PrintKeys();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            print("--- Random Enumerator ---");
            while (random.MoveNext())
            {
                print(random.Current);
            }
            print(Environment.NewLine);
            PrintKeys();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            print("--- Resetting Enumerators ---");
            standardForwards.Reset();
            standardBackwards.Reset();
            yoyo.Reset();
            random.Reset();
            PrintKeys();
        }
    }

    private void PrintKeys()
    {
        print("press S for STANDARD, B for BACKWARDS, Y for YO-YO, R for RANDOM, X to RESET ALL ENUMERATORS");
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Setting the loopCount property to a value of less than 1 will throw an ArgumentOutOfRangeException
