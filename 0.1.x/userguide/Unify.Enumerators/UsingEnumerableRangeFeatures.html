Using Enumerable Range Features

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Enumerable Range functionality is provided by the UnifyEnumerableRange class, that Unify Enumerators expose through the enumerableRange property.

An enumerable range is defined by a start item and an end item; these values are initialised by their index position through the startIndex and endIndex properties.

As the underlying collection is modified the UnifyEnumerableRange will adjust so it is always defined by the items between, and including, the initial start and end items. If either of the start or end items are removed, the new start or end item will become the next item inside the range.

When all items in the range have been removed the UnifyEnumerableRange becomes empty and cannot be used further.

The following animation shows how the UnifyEnumerableRange adjusts to maintain the integrity of the range as the underlying collection is modified.

using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_PartialCollection : MonoBehaviour
{
    private IList<string> list;
    private UnifyForwardsEnumerator<string> forwards;

    void Start()
    {
        list = new List<string>{ "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Mango" };
        forwards = new UnifyForwardsEnumerator<string>(list);
        forwards.enumerableRange.startIndex = 2; // range starts at "Banana"
        forwards.enumerableRange.endIndex = 6; // range ends at "Lemon"

        print("press S to START, R to RESET");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            while (forwards.MoveNext())
            {
                print(forwards.Current);
            }
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            print("--- Resetting Enumerator ---");
            forwards.Reset();
        }
    }
}

To specify an enumerable range that always starts from the first item of a collection, or always ends at the last item of a collection, regardless of any changes made, set the startIndex and/or endIndex properties to -1. This is the default setting.

Looping a Range

Use the loopCount property to specify how many times an enumeration should be performed. This must be a value greater than or equal to 0, otherwise an ArgumentOutOfRangeException will be thrown.

Stepping over a Range

Items can be stepped over during an enumeration through the step property.

This must be a value greater than or equal to 1, otherwise an ArgumentOutOfRangeException will be thrown.
