Unify Enumerator Events

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Unify Enumerators provide the following events:

  • Invalidated - triggered when a Unify Enumerator becomes invalid
  • PassComplete - triggered on each successful pass of a loop
  • Finished - triggered when the Unify Enumerator has finished

All Unify Enumerator events provide a standard non-generic .NET EventHandler delegate for callbacks. Unify Enumerator events do not pass any data to callbacks, currently EventArgs.Empty is passed to subscribers.

Example

This example uses an IUnifyList<T> type, containing the numbers 0-9 as words; "zero", "one", "two", "three", "four", "five", "six", "seven", "eight" and "nine", as the source collection for a UnifyBackwardsEnumerator<T>, with a generic type T of string. A loopCount value of 2 is used to demonstrate when the PassComplete event is triggered, a maxRewindItems value of 1 and an invalidatedAction value of DoNothing is used to prevent the Unify Enumerator from rewinding and calling MoveNext() to show the triggering of the Invalidated event. Error checking is implemented for demonstration purposes. Exceptions are described below.

Attach the script below to a GameObject and press:

  • C to Print the Current item
  • M to Call MoveNext()
  • P to Print the List - the Current item is shown in square brackets and uppercase, e.g. [ZERO]
  • R to Remove the Current item
using System.Text;
using Unify.Collections;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_Events : MonoBehaviour
{
    private IUnifyEnumerator<string> enumerator;
    private IUnifyList<string> list;

    void Awake()
    {
        list = new UnifyList<string> { "zero", one", "two", "three", "four", "five", "six", "seven", "eight", "nine" };

        enumerator = new UnifyBackwardsEnumerator<string>(list)
        {
            loopCount = 2, maxRewindSize = 1, invalidatedAction = UnifyEnumeratorInvalidatedAction.DoNothing
        };

        enumerator.Invalidated += (sender, args) => { print("*** INVALIDATED EVENT TRIGGERED ***"); };
        enumerator.PassComplete += (sender, args) => { print("*** PASS COMPLETE EVENT TRIGGERED ***"); };
        enumerator.Finished += (sender, args) => { print("*** FINISHED EVENT TRIGGERED ***"); };
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.C))
        {
            PrintCurrent();
        }

        if (Input.GetKeyUp(KeyCode.M))
        {
            print("--- MoveNext() ---");
            enumerator.MoveNext();
            PrintCurrent();
        }

        if (Input.GetKeyUp(KeyCode.P))
        {
            PrintList();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            print("--- Remove Current Item ---");
            PrintCurrent();

            if (enumerator.isValid)
            {
                list.Remove(enumerator.Current);
                print("Item Removed");
                PrintList();
                PrintCurrent();
            }
        }
    }

    private void PrintCurrent()
    {
        string current = enumerator.isValid ? enumerator.Current : "INVALID";
        print($"Current Item: [{current.ToUpper()}]");
    }

    private void PrintList()
    {
        int currentIndex = enumerator.isValid ? list.IndexOf(enumerator.Current) : -1;
        StringBuilder sb = new StringBuilder("Current List: ");

        for (int i = 0; i < list.Count; i++)
        {
            sb.Append(currentIndex > -1 && currentIndex == i ? $"[{list[i].ToUpper()}]" : list[i]);

            if (i < list.Count - 1)
            {
                sb.Append(", ");
            }
        }

        print(sb);
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Accessing the Current item when the Unify Enumerator is in an invalid state will throw an ArgumentOutOfRangeException
