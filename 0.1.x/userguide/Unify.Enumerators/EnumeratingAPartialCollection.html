Enumerating a Partial Collection

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Unify Enumerators can enumerate over only part of a collection, the UnifyEnumerableRange class is used to define a start and end index for the part of the collection to be enumerated. These values are specified by the startIndex and endIndex properties respectively, and follow the standard array index convention.

Example

This example demonstrates four ways to use the three built-in Unify Enumerator types; UnifyEnumerator<T>, UnifyYoYoEnumerator<T> and UnifyRandomEnumerator<T>, each with a generic type T of string, to iterate over a collection of fruits; "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry" and "Mango".

  • The standard UnifyEnumerator<T> is used to iterate forwards over the collection, with a startIndex value of 2 and an endIndex value of 9. This produces the enumeration; "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry" and "Mango".

  • The standard UnifyEnumerator<T> is also used to iterate backwards over the collection, with a startIndex value of 5 and an endIndex value of 1. This produces the enumeration; "Plum", "Grape", "Pear", "Banana" and "Orange".

  • The UnifyYoYoEnumerator<T> is used to yo-yo over the collection, with a startIndex value of 0 and an endIndex value of 8. This produces the enumeration; "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Lime", "Lemon", "Plum", "Grape", "Pear", "Banana", "Orange" and "Apple".

  • The UnifyRandomEnumerator<T> iterates randomly over the collection, with a startIndex value of 2 and an endIndex value of 7. The UnifyRandomEnumerator<T> will visit each item in the range only once, producing a randomised enumeration of items between, and including; "Banana" and "Lime".

Attach the script below to a GameObject and press:

  • S to Start the Standard Unify Enumerator
  • B to Start the Standard Unify Enumerator Backwards
  • Y to Start the Yo-Yo Enumerator
  • R to Start the Random Enumerator
  • X to Reset all Enumerators
using System.Collections.Generic;
using Unify.Collections.Generic;
using UnityEngine;

public class Enumerators_SpecifyingAnEnumerableRange : MonoBehaviour
{
    private IList<string> list;
    private IUnifyEnumerator<string> standard;
    private IUnifyEnumerator<string> backwards;
    private IUnifyEnumerator<string> yoyo;
    private IUnifyEnumerator<string> random;

    void Awake()
    {
        list = new List<string> { "Apple", "Orange", "Banana", "Pear", "Grape", "Plum", "Lemon", "Lime", "Cherry", "Mango" };

        standard = new UnifyEnumerator<string>(list, startIndex: 2, endIndex: 9); // range starts at "Banana" and ends at "Mango"

        backwards = new UnifyEnumerator<string>(list, startIndex: 5, endIndex: 1); // range starts at "Plum" and ends at "Orange"

        yoyo = new UnifyYoYoEnumerator<string>(list, startIndex: 0, endIndex: 8); // range starts at "Apple", moves to "Cherry" and then back to "Apple"

        random = new UnifyRandomEnumerator<string>(list, startIndex: 2, endIndex: 7); // range starts at "Banana" and ends at "Lime"

        print("press S for STANDARD, B for BACKWARDS, Y for YO-YO, R for RANDOM, X to RESET ALL ENUMERATORS");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            RunStandard();
        }

        if (Input.GetKeyUp(KeyCode.B))
        {
            RunBackwards();
        }

        if (Input.GetKeyUp(KeyCode.Y))
        {
            RunYoYo();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            RunRandom();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            ResetAll();
        }
    }

    private void RunStandard()
    {
        print("--- Standard Enumerator ---");
        while (standard.MoveNext())
        {
            print(standard.Current);
        }
    }

    private void RunBackwards()
    {
        print("--- Backwards Enumerator ---");
        while (backwards.MoveNext())
        {
            print(backwards.Current);
        }
    }

    private void RunYoYo()
    {
        print("--- YoYo Enumerator ---");
        while (yoyo.MoveNext())
        {
            print(yoyo.Current);
        }
    }

    private void RunRandom()
    {
        print("--- Random Enumerator ---");
        while (random.MoveNext())
        {
            print(random.Current);
        }
    }

    private void ResetAll()
    {
        print("--- Resetting Enumerators ---");
        standard.Reset();
        backwards.Reset();
        yoyo.Reset();
        random.Reset();
    }
}

Things to consider

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Setting the startIndex property to a value of less than 0 will throw an ArgumentOutOfRangeException
  • Setting the endIndex property to a value of less than 0 will throw an ArgumentOutOfRangeException
  • Setting the step property to a value of less than 1 will throw an ArgumentOutOfRangeException
  • Setting the loopCount property to a value of less than 1 will throw an ArgumentOutOfRangeException
