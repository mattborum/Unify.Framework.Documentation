Unify Coroutine Events

Unify Coroutines provide the following events:

  • Started - triggered when a Unify Coroutine is successfully started with the Start() method
  • Paused - triggered when a Unify Coroutine is successfully paused with the Pause() method
  • Resumed - triggered when a Unify Coroutine is successfully resumed with the Resume() method
  • Stopped - triggered when a Unify Coroutine is successfully stopped with the Stop() method
  • Cancelled - triggered when a Unify Coroutine successfully is cancelled with the Cancel() method
  • PassComplete - triggered each time a looping Unify Coroutine successfully completes a pass of the loop
  • Finished - triggered when a Unify Coroutine successfully finishes (for repeating Unify Coroutines, this is triggered after all passes have successfully completed)

Subscribing to Unify Coroutine events

There are three ways to subscribe to Unify Coroutine events; programmatically, anonymous methods and lambda expressions:

Programmatically

myUnifyCoroutine.Started += StartedEventHandler;

private void StartedEventHandler(object sender, UnifyCoroutineEventArgs e)
{
	print("* STARTED EVENT HANDLER *");
}

Anonymous Method

myUnifyCoroutine.Started += delegate(object sender, UnifyCoroutineEventArgs e)
{
	print("* STARTED EVENT HANDLER *");
};

Lambda Expression

myUnifyCoroutine.Started += (sender, e) =>
{
	print("* STARTED EVENT HANDLER *");
};

Unsubscribing from Unify Coroutine events

Programmatically

myUnifyCoroutine.Started -= StartedEventHandler;

private void StartedEventHandler(object sender, UnifyCoroutineEventArgs e)
{
    print("* STARTED EVENT HANDLER *");
}

If a Unify Coroutine event is subscribed to via either an anonymous method or lambda expression, then it is not possible to unsubscribe from that event unless using a reference to the original method which was used to subscribe:

Anonymous Method

private EventHandler<UnifyCoroutineEventArgs> myUnifyCoroutineEventDelegate;
myUnifyCoroutineEventDelegate = delegate { print("* STARTED EVENT HANDLER *"); };

myUnifyCoroutine.Started += myUnifyCoroutineEventDelegate;  // Subscribe
myUnifyCoroutine.Started -= myUnifyCoroutineEventDelegate;  // Unsubscribe

Lambda Expression

private EventHandler<UnifyCoroutineEventArgs> myUnifyCoroutineEventDelegate;
myUnifyCoroutineEventDelegate = (sender, args) => { print(* STARTED EVENT HANDLER *); };

myUnifyCoroutine.Started += myUnifyCoroutineEventDelegate; // Subscribe
myUnifyCoroutine.Started -= myUnifyCoroutineEventDelegate; // Unsubscribe

Event Args

Unify Coroutines provide a custom data base class named UnifyCoroutineEventArgs, an instance of which is passed to event handlers. This base class holds no data, it is provided as a way for you to create your own custom EventArgs classes.

Example

This Unify Coroutine example counts from 1 to 20, printing all even numbers to the Unity Console at 1 second intervals. The Unify Coroutine is repeated 4 times with a 1 second delay between each pass of the loop.
Here, the loopCount and passDelay properties are initialised through the Unify Coroutine constructor.
Also, keepAlive has been set to true to allow restarting and failSilently has been set to false to allow you to trigger exceptions as described below.

This example shows the three ways you can subscribe to Unify Coroutine events. The Pause() method is used to demonstrate unsubscribing from a Unify Coroutine event using the programmatical method described above.

Attach the script below to a GameObject and press:

  • S to Start
  • P to Pause
  • R to Resume
  • X to Stop
  • C to Cancel
  • E to subscribe to/unsubscribe from the Paused event
using System;
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Events : MonoBehaviour
{
    private UnifyCoroutine myUnifyCoroutine;
    private bool pausedSubscribeToggle = false;

    private EventHandler<UnifyCoroutineEventArgs> pausedEventDelegate;
    private EventHandler<UnifyCoroutineEventArgs> passCompleteEventDelegate;

    void Start()
    {
        myUnifyCoroutine = new UnifyCoroutine(WaitAndPrint(), name: "Grace", loopCount: 4, passDelay: 1f) { failSilently = false, keepAlive = true };

        // Subscribe to the Started event programmatically.
        myUnifyCoroutine.Started += StartedEventHandler;

        // Subscribe to the Paused event programmatically.
        myUnifyCoroutine.Paused += PausedEventHandler;
        myUnifyCoroutine.Paused += pausedEventDelegate;

        // Subscribe to the Resumed event using an anonymous method.
        myUnifyCoroutine.Resumed += delegate(object sender, UnifyCoroutineEventArgs e)
        {
            print("* RESUMED EVENT HANDLER *");
        };

        // Subscribe to the Stopped event using an anonymous method.
        myUnifyCoroutine.Stopped += delegate(object sender, UnifyCoroutineEventArgs e)
        {
            print("* STOPPED EVENT HANDLER *");
        };

        // Subscribe to the Cancelled event using a lambda expression.
        myUnifyCoroutine.Cancelled += (sender, e) =>
        {
            print("* CANCELLED EVENT HANDLER *");
        };

        // Subscribe to the PassComplete event using a lambda expression through a referenced delegate
        passCompleteEventDelegate = (sender, args) => { print("* PASS COMPLETE EVENT HANDLER*");};
        myUnifyCoroutine.PassComplete += passCompleteEventDelegate;

        // Subscribe to the Finished event using a lambda expression.
        myUnifyCoroutine.Finished += (sender, e) =>
        {
            print("* FINISHED EVENT HANDLER *");
        };

        print("press S to START, P to PAUSE, R to RESUME, X to STOP, C to CANCEL, E to subscribe to/unsubscribe from PAUSED event");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            myUnifyCoroutine.Start();
        }

        if (Input.GetKeyUp(KeyCode.P))
        {
            myUnifyCoroutine.Pause();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            myUnifyCoroutine.Resume();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            myUnifyCoroutine.Stop();
        }

        if (Input.GetKeyUp(KeyCode.C))
        {
            myUnifyCoroutine.Cancel();
        }

        if (Input.GetKeyUp(KeyCode.E))
        {
            if (pausedSubscribeToggle)
            {
                myUnifyCoroutine.Paused += PausedEventHandler;
                myUnifyCoroutine.Paused += pausedEventDelegate;

                print("* SUBSCRIBED TO PAUSED EVENT *");
            }
            else
            {
                myUnifyCoroutine.Paused -= PausedEventHandler;
                myUnifyCoroutine.Paused -= pausedEventDelegate;

                print("* UNSUBSCRIBED FROM PAUSED EVENT *");
            }

            pausedSubscribeToggle = !pausedSubscribeToggle;
        }
    }

    IEnumerator WaitAndPrint()
    {
        for (int i = 1; i <= 20; i++)
        {
            if (i % 2 == 0)
            {
                print(i);
                yield return new WaitForSecondsRealtime(1f);
            }
        }
    }

    private void StartedEventHandler(object sender, UnifyCoroutineEventArgs e)
    {
        print("* STARTED EVENT HANDLER *");
    }

    private void PausedEventHandler(object sender, UnifyCoroutineEventArgs e)
    {
        print("* PAUSED EVENT HANDLER *");
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter

Unify does not throw exceptions specifically for Unify Coroutine events, however exceptions will be thrown from other operations:

  • Creating and Starting
  • Pausing and Resuming
  • Stopping and Restarting
  • Cancelling
