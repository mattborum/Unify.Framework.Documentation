Stopping and Restarting a Unify Coroutine Collection

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

Unify Coroutine Collections can be stopped and restarted in the same way a Unify Coroutine can be stopped and restarted.
To enable this, the keepAlive property of each Unify Coroutine in the Unify Coroutine Collection must be set to true before starting the Unify Coroutine Collection for the first time.

Example

This example adds two Unify Coroutines to a Unify Coroutine Collection; CountToSixty() and PrintAlphabet(). The keepAlive property for both Unify Coroutines has been set to true to allow restarting and failSilently has been set to false to allow exceptions to be triggered as described below.
The name property has been set to 'Joachim Ante' to demonstrate how Unify Coroutine instances are identified in exception messages.

Attach the script below to a GameObject and press:

  • S to Start
  • X to Stop
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Collection_StoppingAndRestarting : MonoBehaviour
{
    private UnifyCoroutineCollection collection;
    private Random random = new Random();

    void Awake()
    {
        collection = new UnifyCoroutineCollection(name: "Joachim Ante") { failSilently = false };

        collection.Add(new UnifyCoroutine(CountToSixty()) { keepAlive = true } );
        collection.Add(new UnifyCoroutine(PrintAlphabet()) { keepAlive = true });

        print("press S to START, X to STOP");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("--- Start ---");
            collection.Start();
        }

        if (Input.GetKeyUp(KeyCode.X))
        {
            print("--- Stop ---");
            collection.Stop();
        }
    }

    private IEnumerator CountToSixty()
    {
        for (int i = 1; i <= 60; i++)
        {
            print(i);
            yield return new WaitForSecondsRealtime(0.75f);
        }
    }

    private IEnumerator PrintAlphabet()
    {
        for (int i = 65; i <= 90; i++)
        {
            print((char)i);
            yield return new WaitForSecondsRealtime(1f);
        }
    }

}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter of a Unify Coroutine Collection constructor, collection initialiser or Add() method

Exceptions will be thrown under the following circumstances, only if failSilently is false:

  • Starting a Unify Coroutine Collection that is in a state other than Stopped or Ready
  • Stopping a Unify Coroutine Collection that is in a state other than Running
