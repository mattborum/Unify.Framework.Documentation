Introduction to Unify Coroutines

What's the difference between Pause, Stop and Cancel?

Pause()

Temporarily halts execution until the Resume() method is called. Execution resumes from the point is was paused.

Stop()

Halts execution. The Unify Coroutine can be restarted with the Start() method. Execution starts from the beginning.

Cancel()

Permanently halts execution. The Unify Coroutine cannot be resumed or restarted.
