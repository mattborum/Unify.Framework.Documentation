Looping a Unify Coroutine Collection

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

Looping a Unify Coroutine Collection is achieved through the loopCount property, the same way a Unify Coroutine is looped; the loopCount property specifies how many times the Unify Coroutine Collection should be repeated. One pass of a Unify Coroutine Collection loop is considered completed when all Unify Coroutines in the collection are finished. Only Unify Coroutines in the collection whose keepAlive property has a value of true will be repeated.

The passDelay property is used to delay the starting of the next pass in the loop.
