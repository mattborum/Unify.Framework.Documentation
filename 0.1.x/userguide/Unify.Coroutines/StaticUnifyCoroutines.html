Static Unify Coroutines

Unify Coroutines provide a static Start() method that enables the starting of a Unify Coroutine without needing to create an instance.
Static Unify Coroutines still offer delayed start and looping functionality through the startDelay, loopCount and passDelay properties, values for which can be passed into the static Start() method as parameters.
The static Start() method also offers subscription to the Finished and passComplete events through the finishedEventHandler and [passCompleteEventHandler] parameters respectively.

The static Start() method will return true if the Unify Coroutine was started successfully, otherwise it will return false.

bool started = UnifyCoroutine.Start(SayHello(), startDelay: 1f, loopCount: 2, passDelay: 1f,
                                    finishedEventHandler: (s, e) =>
                                    {
                                        print("* Static Unify Coroutine FINISHED *");
                                    } );

IEnumerator SayHello()
{
    print("Hello, Unify!");
    yield return null;
}

Example One

This Unify Coroutine example uses a standard Unify Coroutine to count from 1 to 10 and print the square of each value to the Unity Console at 1 second intervals. The Unify Coroutine has a startDelay of 1 second, a loopCount of 3 and a passDelay of 2.5 seconds. The static Start() method is used to begin execution of the Unify Coroutine. Here, the Finished event handler is subscribed to using a lambda expression through the finishedEventHandler parameter, failSilently has been set to false to allow you to trigger exceptions as described below, and the name property has been set to 'William' to demonstrate how Unify Coroutine instances are identified in exception messages.

Attach the script below to a GameObject and press:

  • S to Start the Unify Coroutine
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_StaticStart : MonoBehaviour
{
    void Start()
    {
        print("press S to START");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("* Starting Unify Coroutine *");

            UnifyCoroutine.Start(SquaresOfOneToTen(), startDelay: 1f, loopCount: 3, passDelay: 2.5f, name: "William",
                passCompleteEventHandler: (s, e) =>
                {
                    print("* PASS COMPLETE *");
                },
                finishedEventHandler: (s, e) =>
                {
                    print("* Static Unify Coroutine FINISHED *");
                });
        }
    }

    IEnumerator SquaresOfOneToTen()
    {
        for (int i = 1; i <= 10; i++)
        {
            print($"sqr({i}) = " + i * i);
            yield return new WaitForSecondsRealtime(1f);
        }
    }
}

Example Two

This Unify Coroutine example shows how to return a value from a static Unify Coroutine. It is the same example as above but uses a generic UnifyCoroutine<T> to return the value, rather than print it to the Unity Console from within the Coroutine. The return value is picked up by the ValueReturned event handler which is passed as a lambda expression through the constructor.

Attach the script below to a GameObject and press:

  • S to Start the Unify Coroutine
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_StaticStartReturnValue : MonoBehaviour
{
    void Start()
    {
        print("press S to START");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("* Starting Unify Coroutine *");

            UnifyCoroutine<int>.Start(SquaresOfOneToTen(), startDelay: 1f, loopCount: 3, passDelay: 2.5f, name: "William",
                valueReturnedHandler: (s, e) =>
                {
                    print($"sqr({e.returnValue}) = {e.returnValue * e.returnValue}");
                },
                passCompleteEventHandler: (s, e) =>
                {
                    print("* PASS COMPLETE *");
                },
                finishedEventHandler: (s, e) =>
                {
                    print("* Static Unify Coroutine FINISHED *");
                });
        }
    }

    IEnumerator SquaresOfOneToTen()
    {
        for (int i = 1; i <= 10; i++)
        {
            yield return i;
            yield return new WaitForSecondsRealtime(1f);
        }
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter
  • Passing a value of less than 0 to the startDelay property or parameter
  • Passing a value of less than -1 to the loopCount property or paramter
  • Passing a value of less than 0 to the passDelay property or parameter
