Looping a Unify Coroutine

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

Unify Coroutines provide a loopCount property to specify the number times the Coroutine should be executed. To enable this functionality, either the loopCount property or the keepAlive property must be initialised before starting the Unify Coroutine for the first time.

Unify Coroutines also provide a passDelay property to allow a delay between each pass (execution of the Unify Coroutine).

The loopCount and passDelay properties can be set in two ways:

Pass loopCount and/or passDelay as parameters to the constructor:

UnifyCoroutine myUnifyCoroutine = new UnifyCoroutine(CountToTen(), loopCount: 3, passDelay: 1.5f);

private IEnumerator CountToTen()
{
    for (int i = 1; i <= 10; i++)
    {
        print($"counting... {i}");
        yield return new WaitForSecondsRealtime(0.25f);
    }
}

Set the loopCount and/or passDelay properties directly:

myUnifyCoroutine.loopCount = 3;
myUnifyCoroutine.passDelay = 1.5f; // 1.5 seconds delay between each pass

Events

Unify Coroutines provide a passComplete event that is triggered after each pass of the loop.

Example

This Unify Coroutine example calculates the first 10 numbers in the Fibonacci Sequence and prints each value to the Unity Console at intervals of 0.3 seconds.
The Unify Coroutine is repeated 3 times with a 2.5 second delay between each pass of the loop. Here, the loopCount and passDelay properties are initialised through the Unify Coroutine constructor.
Also, keepAlive has been set to true to allow looping and failSilently has been set to false to allow you to trigger exceptions as described below. The name property has been set to 'Ginny' to demonstrate how Unify Coroutine instances are identified in exception messages.

Attach the script below to a GameObject and press:

  • S to Start the Unify Coroutine
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Looping : MonoBehaviour
{
    private UnifyCoroutine myUnifyCoroutine;

    void Start()
    {
        myUnifyCoroutine = new UnifyCoroutine(Fibonacci(), loopCount: 3, passDelay: 2.5f, name: "Ginny");
        myUnifyCoroutine.failSilently = true;
        myUnifyCoroutine.keepAlive = false;

        myUnifyCoroutine.PassComplete += MyUnifyCoroutine_PassComplete;

        print("press S to START");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            myUnifyCoroutine.Start();
        }
    }

    private IEnumerator Fibonacci()
    {
        int a = 0;
        int b = 1;

        for (int i = 0; i < 10; i++)
        {
            for (int j = 0; j < i; j++)
            {
                int temp = a;
                a = b;
                b = temp + b;
            }

            print(a); 

            yield return new WaitForSecondsRealtime(0.3f);
        }
    }

    private void MyUnifyCoroutine_PassComplete(object sender, UnifyCoroutineEventArgs e)
    {
        print("* PASS COMPLETE *");
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter
  • Passing a value of less than -1 to the loopCount property or parameter
  • Passing a value of less than 0 to the passDelay property or parameter
