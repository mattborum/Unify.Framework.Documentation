Pausing and Resuming a Unify Coroutine Collection

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

Unify Coroutine Collections can be paused and resumed in the same way a Unify Coroutine can be paused and resumed.

Example

Here, failSilently has been set to false to allow you to trigger exceptions as described below. The name property has been set to 'Nicholas Francis' to demonstrate how Unify Coroutine Collection instances are identified in exception messages.

Attach the script below to a GameObject and press:

  • S to Start
  • P to Pause
  • R to Resume
using System.Collections;
using Random = System.Random;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Collection_PausingAndResuming : MonoBehaviour
{
    private UnifyCoroutineCollection collection;
    private Random random = new Random();

    void Awake()
    {
        collection = new UnifyCoroutineCollection(name: "Nicholas Francis");
        collection.failSilently = false;

        collection.Add(CountToSixty());
        collection.Add(PrintRandomLetter());
        collection.Add(SayHello());

        print("press S to START, P to PAUSE, R to RESUME");
    }
    
    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("--- Start ---");
            collection.Start();
        }

        if (Input.GetKeyUp(KeyCode.P))
        {
            print("--- Paused ---");
            collection.Pause();
        }

        if (Input.GetKeyUp(KeyCode.R))
        {
            print("--- Resumed ---");
            collection.Resume();
        }
    }

    private IEnumerator CountToSixty()
    {
        for (int i = 1; i <= 60; i++)
        {
            print(i);
            yield return new WaitForSecondsRealtime(1f);
        }
    }

    private IEnumerator PrintRandomLetter()
    {
        for (int i = 0; i <= 30; i++)
        {
            int c = random.Next(65, 91);
            print((char)c);
            yield return new WaitForSecondsRealtime(2f);
        }
    }

    private IEnumerator SayHello()
    {
        for (int i = 0; i < 10; i++)
        {
            print("Hello!");
            yield return new WaitForSecondsRealtime(5f);
        }
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter of a Unify Coroutine Collection constructor, collection initialiser or Add() method

Exceptions will be thrown under the following circumstances, only if failSilently is false:

  • Pausing a Unify Coroutine Collection that is in a state other than Running
  • Resuming a Unify Coroutine Collection that is in a state other than Paused

This message tells us the exception was caused by calling the Resume() method on a Unify Coroutine Collection that was already in a Paused state, and that 'Nicholas Francis' was the name of the offending Unify Coroutine Collection instance.
