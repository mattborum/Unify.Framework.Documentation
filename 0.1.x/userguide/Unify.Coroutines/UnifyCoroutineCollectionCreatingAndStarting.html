Creating and Starting a Unify Coroutine Collection

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

This example shows how to create and start a UnifyCoroutineCollection. It demonstrates adding Unify Coroutines using Collection Initialisers and also through the Add() method.

Attach the script below to a GameObject and press:

  • S to Start
using System;
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Collection_CreatingAndStarting : MonoBehaviour
{
    private UnifyCoroutineCollection collection;
    private string[] days;

    void Awake()
    {
        days = new[] { "Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday" };

        collection = new UnifyCoroutineCollection { PrintAlpha(), PrintNumeric() };
        collection.Add(PrintTime());
        print("press S to START");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            collection.Start();
        }
    }

    private IEnumerator PrintAlpha()
    {
        for (int i = 0; i < days.Length; i++)
        {
            print(days[i]);
            yield return new WaitForSecondsRealtime(0.75f);
        }
    }

    private IEnumerator PrintNumeric()
    {
        for (int i = 1; i <= 10; i++)
        {
            print(i);
            yield return new WaitForSecondsRealtime(1f);
        }
    }

    private IEnumerator PrintTime()
    {
        for (int i = 0; i < 12; i++)
        {
            print(DateTime.Now.ToString("HH:mm:ss"));
            yield return new WaitForSecondsRealtime(2f);
        }
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter of the Add() method or Collection Initialiser.

Exceptions will be thrown under the following circumstances, only if failSilently is false:

  • Starting a Unify Coroutine Collection that is in a Running state
