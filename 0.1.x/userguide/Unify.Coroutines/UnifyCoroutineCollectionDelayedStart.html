Delayed Start of a Unify Coroutine Collection

Namespace: Unify.Coroutines
Assembly: Unify.Coroutines.dll

To delay the start of a UnifyCoroutineCollection, use the startDelay property in the same way as a Unify Coroutine delayed start. The Unify Coroutine Collection's startDelay value delays the start of all Unify Coroutines in the UnifyCoroutineCollection, it does not change the startDelay value for each Unify Coroutine in the collection; so once the Unify Coroutine Collection's startDelay has expired, then the startDelay for each individual Unify Coroutine in the collection will be applied. For example, if a UnifyCoroutineCollection has a startDelay of 1 second and a Unify Coroutine in the collection has a startDelay of 2 seconds, then it will be a total of 3 seconds before Unify starts that Unify Coroutine.

Example

This example uses two Unify Coroutines; SayHello(), which has a startDelay of 1 second, and SayWorld(), which has a startDelay of 3 seconds. The UnifyCoroutineCollection is given a startDelay of 2 seconds; this means the SayHello() Unify Coroutine will start after 3 seconds and the SayWorld() Unify Coroutine will start after 5 seconds.

A Collection Initialiser is used to demonstrate how to populate a UnifyCoroutineCollection with a Unify Coroutine target and it's startDelay value via the Unify Collection's constructor.

The name property has been set to 'John Riccitiello' to demonstrate how Unify Coroutine Collection instances are identified in exception messages and failSilently has been set to false to allow exceptions to be triggered, as described below.

Attach the script below to a GameObject and press:

  • S to Start
using System.Collections;
using Unify.Coroutines;
using UnityEngine;

public class Coroutines_Collection_DelayedStart : MonoBehaviour
{

    private UnifyCoroutineCollection collection;

    void Awake()
    {
        collection = new UnifyCoroutineCollection { { SayHello(), 1f }, { SayWorld(), 3f } };
        collection.startDelay = 2f;

        print("press S to START");
    }

    void Update()
    {
        if (Input.GetKeyUp(KeyCode.S))
        {
            print("--- Starting Unify Coroutine Collection ---");
            collection.Start();
        }
    }

    private IEnumerator SayHello()
    {
        yield return new WaitForSecondsRealtime(1.5f);
        print("Hello");
    }

    private IEnumerator SayWorld()
    {
        yield return new WaitForSecondsRealtime(3f);
        print("World");
    }
}

Exceptions

Exceptions will always be thrown under the following circumstances:

  • Passing a null value to the target parameter of a Unify Coroutine constructor, collection initialiser or Add() method
  • Passing a value of less than 0 to the startDelay parameter

Exceptions will be thrown under the following circumstances, only if failSilently is false:

  • Starting a Unify Coroutine Collection that is in a Running state
