Unify Collection Events

Unify Collections provide the following events:

  • CollectionChanged and CollectionChanged<T> - triggered when an item is added, inserted or removed
  • CollectionCleared - triggered when either a generic Unify Collection's Clear() method, or non-generic Unify Collection's Clear() method is called

This functionality is exposed through the IUnifyObservableCollection and IUnifyObservableCollection<T> interface types, which all Unify Collections implement.

Subscribing to Unify Collection events

There are three ways to subscribe to Unify Collection events; programmatically, anonymous methods and lambda expressions:

Programmatically

myUnifyCollection.CollectionChanged += CollectionChangedEventHandler;

private void CollectionChangedEventHandler(object sender, UnifyCollectionChangedEventArgs e)
{
	print("* UNIFY COLLECTION CHANGED EVENT HANDLER *");
}

Anonymous Method

myUnifyCollection.CollectionChanged += delegate(object sender, UnifyCollectionChangedEventArgs e)
{
	print("* UNIFY COLLECTION CHANGED EVENT HANDLER *");
};

Lambda Expression

myUnifyCollection.CollectionChanged += (sender, e) =>
{
	print("* UNIFY COLLECTION CHANGED EVENT HANDLER *");
};

If a Unify Collection event is subscribed to via either an anonymous method or lambda expression, then it is not possible to unsubscribe from that event unless using a reference to the original method which was used to subscribe:

Anonymous Method

private EventHandler<UnifyCollectionChangedEventArgs> myUnifyCollectionChangedEventDelegate;
myUnifyCollectionChangedEventDelegate = delegate { print("* UNIFY COLLECTION CHANGED EVENT HANDLER *"); };

myUnifyCollection.CollectionChanged += myUnifyCollectionChangedEventDelegate;  // Subscribe
myUnifyCollection.CollectionChanged -= myUnifyCollectionChangedEventDelegate;  // Unsubscribe

Lambda Expression

private EventHandler<UnifyCollectionChangedEventArgs> myUnifyCollectionChangedEventDelegate;
myUnifyCollectionChangedEventDelegate = (sender, args) => { print("* UNIFY COLLECTION CHANGED EVENT HANDLER *"); };

myUnifyCollection.CollectionChanged += myUnifyCollectionChangedEventDelegate;  // Subscribe
myUnifyCollection.CollectionChanged -= myUnifyCollectionChangedEventDelegate;  // Unsubscribe

EventArgs

Unify Collections provide generic UnifyCollectionChangedEventArgs<T> and non-generic UnifyCollectionChangedEventArgs classes, an instance of which is passed to event handlers. This class provides information about the change to the collection; the action property is a UnifyCollectionChangedAction type, that specifies the action causing the change; the item property, of type T for generic collections and of type object for non-generic collections, contain the item responsible for the change and the index property is the index position of that item.
