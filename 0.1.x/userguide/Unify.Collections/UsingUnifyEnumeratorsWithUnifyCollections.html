Using Unify Enumerators with Unify Collections

Unify Enumerators are designed to work seamlessly with Unify Collections and .NET.

The foreach statement works with Unify in the same way as it does with .NET:

UnifyList<string> list = new UnifyList<string> { "Hearts", "Diamonds", "Spades", "Clubs" };

foreach (string s in list)
{
    print(s);
}

/*  Produces the following output:
 *
 *  Hearts
 *  Diamonds
 *  Spades
 *  Clubs
 */

The GetEnumerator() method also works in the same way as .NET:

UnifyList<string> list = new UnifyList<string> { "Hearts", "Diamonds", "Spades", "Clubs" };
IUnifyEnumerator<string> enumerator = list.GetEnumerator();

while (enumerator.MoveNext())
{
    print(enumerator.Current);
}

/*  Produces the following output:
 *
 *  Hearts
 *  Diamonds
 *  Spades
 *  Clubs
 */

Enumerable collections in Unify provide an additional method, SetEnumerator(), to specify the default enumerator for the collection. An instance of the current default enumerator will be returned by the GetEnumerator() method and is also the one used by the foreach statement. The SetEnumerator() method takes a value from the enum class, UnifyEnumeratorType, and can be any one of the built-in Unify Enumerator types; Forwards, Backwards, YoYo or Random.

UnifyList<string> list = new UnifyList<string> { "Hearts", "Diamonds", "Spades", "Clubs" };

list.SetEnumerator(UnifyEnumeratorType.Backwards); // set the default enumerator to Backwards

print("foreach");
foreach (string s in list)
{
    print($"foreach: {s}");
}

IUnifyEnumerator<string> enumerator = list.GetEnumerator();

print(Environment.NewLine);
print("while");
while (enumerator.MoveNext())
{
    print($"while: {enumerator.Current}");
}

/*  Produces the following output:
 *
 * foreach: Clubs
 * foreach: Spades
 * foreach: Diamonds
 * foreach: Hearts
 *
 * while: Clubs
 * while: Spades
 * while: Diamonds
 * while: Hearts
 */

The SetEnumerator() method also provides overloads to specify values for the enumerator's EnumerableRange; these overloaded methods allow startIndex, endIndex, loopCount and step properties to be specified. This code snippet uses a list of computer game characters; "Mario", "Sonic", "Kirby", "Yoshi", "Luigi", "Master Chief", "Solid Snake", "Pikachu", "Bowser", "Link", and specifies the enumerable range with a startIndex of 2, an endIndex of 6 and a step value of 2. This gives an enumerable range with the values; "Kirby", "Luigi", "Solid Snake" which the UnifyRandomEnumerator<T> will use to iterate over:

UnifyList<string> list = new UnifyList<string> { "Mario", "Sonic", "Kirby", "Yoshi", "Luigi", "Master Chief", "Solid Snake", "Pikachu", "Bowser", "Link" };

list.SetEnumerator(UnifyEnumeratorType.Random, 2, 6, 2);

foreach (string s in list)
{
	print($"foreach: {s}");
}

The Unify Enumerator's GetEnumerator() method also provides overloads. The GetEnumerator(UnifyEnumeratorType) overload returns an instance of any of the built-in Unify Enumerator types, via the UnifyEnumeratorType parameter, without changing the collection's default enumerator. It also allows an enumerable range to be defined in the same way as the SetEnumerator() method, again this does not change the default enumerator's enumerable range values:

IUnifyList<string> list = new UnifyList<string> { "Hamlet", "Macbeth", "Romeo and Juliet", "Julius Caesar", "A Midsummer Night's Dream" };
        
// get a Backwards enumerator instance for 'list', leaving the default enumerator unchanged
// set the enumerable range for the backwards enumerator as all items from, and including, index 2 to index 5; "Romeo and Juliet", "Julius Caesar", "A Midsummer Night's Dream"
IUnifyEnumerator<string> enumerator = list.GetEnumerator(UnifyEnumeratorType.Backwards, 2, 5);

print("Default Enumerator:");

foreach (string s in list)
{
    print(s);
}

print(Environment.NewLine);
print("GetEnumerator(UnifyEnumeratorType.Backwards, 2, 5)");

while (enumerator.MoveNext())
{
    print(enumerator.Current);
}

/* Produces the following output:
 *
 * Default Enumerator:
 * Hamlet
 * Macbeth
 * Romeo and Juliet
 * Julius Caesar
 * A Midsummer Night's Dream
 *
 * GetEnumerator(UnifyEnumeratorType.Backwards, 2, 5)
 * A Midsummer Night's Dream
 * Julius Caesar
 * Romeo and Juliet
 *
 */
