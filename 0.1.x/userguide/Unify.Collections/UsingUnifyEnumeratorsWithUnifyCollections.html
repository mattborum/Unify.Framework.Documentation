Using Unify Enumerators with Unify Collections

Namespace: Unify.Collections, Unify.Collections.Generic
Assembly: Unify.Collections.dll

Unify Enumerators are designed to work seamlessly with Unify Collections and .NET.

The foreach statement works with Unify in the same way as it does with .NET:

UnifyList<string> list = new UnifyList<string> { "Untitled Goose Game", "Hearthstone", "Subnautica", "Kerbal Space Program" };

foreach (string s in list)
{
    print(s);
}

/*  Produces the following output:
 *
 *  Untitled Goose Game
 *  Hearthstone
 *  Subnautica
 *  Kerbal Space Program
 */

The GetEnumerator() method also works with Unify in the same way as .NET:

UnifyList<string> list = new UnifyList<string> { "Untitled Goose Game", "Hearthstone", "Subnautica", "Kerbal Space Program" };
IUnifyEnumerator<string> enumerator = list.GetEnumerator();

while (enumerator.MoveNext())
{
    print(enumerator.Current);
}

/*  Produces the following output:
 *
 *  Untitled Goose Game
 *  Hearthstone
 *  Subnautica
 *  Kerbal Space Program
 */

The foreach and GetEnumerator() examples above both use a UnifyEnumerator. An instance of the current default enumerator is returned by the GetEnumerator() method, which also provides the foreach statement with a Unify Enumerator. The default Unify Enumerator for any IUnifyEnumerable type is a UnifyForwardsEnumerator instance.

To change which Unify Enumerator is used, IUnifyEnumerable types in Unify provide a SetEnumerator(UnifyEnumeratorType) method, the UnifyEnumeratorType parameter is used to specify the UnifyEnumerator, and can be any one of the built-in Unify Enumerator types; Forwards, Backwards, YoYo or Random.

UnifyList<string> list = new UnifyList<string> { "Hollow Knight", "Cuphead", "Firewatch", "Rust", "Monument Valley" };

list.SetEnumerator(UnifyEnumeratorType.Backwards); // set the default enumerator to Backwards

print("foreach");
foreach (string s in list)
{
    print($"foreach: {s}");
}

IUnifyEnumerator<string> enumerator = list.GetEnumerator();

print(Environment.NewLine);
print("while");
while (enumerator.MoveNext())
{
    print($"while: {enumerator.Current}");
}

/*  Produces the following output:
 *
 * foreach: Monument Valley
 * foreach: Rust
 * foreach: Firewatch
 * foreach: Cuphead
 * foreach: Hollow Knight
 *
 * while: Monument Valley
 * while: Rust
 * while: Firewatch
 * while: Cuphead
 * while: Hollow Knight
 */

The SetEnumerator() method also provides an overload, SetEnumerator(UnifyEnumeratorType, UnifyEnumerableRange), that allows the Unify Enumerator's EnumerableRange values to be set at the same time.

This code snippet uses a list of computer game characters; "Mario", "Sonic", "Kirby", "Yoshi", "Luigi", "Master Chief", "Solid Snake", "Pikachu", "Bowser", "Link", and specifies the enumerable range with a startIndex of 2, an endIndex of 6 and a step value of 2. The loopCount property is not specified so will remain at it's default value of 1. This gives an enumerable range containing the values; "Kirby", "Luigi" and "Solid Snake", which a UnifyRandomEnumerator<T> instance will use to iterate over:

UnifyList<string> list = new UnifyList<string> { "Mario", "Sonic", "Kirby", "Yoshi", "Luigi", "Master Chief", "Solid Snake", "Pikachu", "Bowser", "Link" };

list.SetEnumerator(UnifyEnumeratorType.Random, new UnifyEnumerableRange(startIndex:2, endIndex:6, step:2));

foreach (string s in list)
{
	print($"foreach: {s}");
}

/* Produces output similar to the following:
 *
 * foreach: Luigi
 * foreach: Solid Snake
 * foreach: Kirby
 */

The Unify Enumerator's GetEnumerator() method also provides an overload, GetEnumerator(UnifyEnumeratorType), that returns an instance of any of the built-in Unify Enumerator types, via the UnifyEnumeratorType parameter, without changing the collection's default enumerator or EnumerableRange values:

IUnifyList<string> list = new UnifyList<string> { "My Friend Pedro", "Pillars of Eternity", "Cities: Skylines", "Ori and the Blind Forest", "Risk of Rain 2" };

IUnifyEnumerator<string> enumerator = list.GetEnumerator(UnifyEnumeratorType.Backwards);
enumerator.startIndex = 1;
enumerator.endIndex = 3;

print("Default Enumerator");
print("------------------");

foreach (string s in list)
{
	print(s);
}

print(Environment.NewLine);
print("GetEnumerator(UnifyEnumeratorType.Backwards)");
print("startIndex = 1, endIndex = 3");
print("--------------------------------------------");

while (enumerator.MoveNext())
{
	print(enumerator.Current);
}

enumerator.Dispose();

/* Produces the following output:
 *
 * Default Enumerator
 * ------------------
 * My Friend Pedro
 * Pillars of Eternity
 * Cities: Skylines
 * Ori and the Blind Forest
 * Risk of Rain 2
 *
 * GetEnumerator(UnifyEnumeratorType.Backwards)
 * startIndex = 1, endIndex = 3
 * --------------------------------------------
 * Ori and the Blind Forest
 * Cities: Skylines
 * Pillars of Eternity
 */
